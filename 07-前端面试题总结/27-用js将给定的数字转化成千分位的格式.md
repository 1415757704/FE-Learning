## 用js将给定的数字转化成千分位的格式，如把“10000”转化成“10,000”，并考虑到性能方面的因素。

使用正则表达式

// 解法1: 只对整数有效,若有小数会出问题
```
function parseNum(num){  
    var reg=/(?=(?!\b)(\d{3})+$)/g;  
    return String(num).replace(reg, ',');  
}  
console.log(parseNum(10000121213));
```

//解法2: 对小数也起作用
```
var n="45854236.558423";
var re = /(\d{1,3})(?=(\d{3})+(?:\.))/g;
console.log(n.replace(re, "$1,"));
```
