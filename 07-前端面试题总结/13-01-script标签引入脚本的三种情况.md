### <script> 标签引入脚本有三种情况：

#### 1. 立即执行
```
<script src="a.js">
<script src="b.js">
顺序：保证先后顺序。解析：HTML 解析器遇到它们时，将阻塞（取停止解析），
待脚本下载完成并执行后，继续解析标签之后的文档。
```

#### 2.推迟执行
```
<script defer src="a.js">
<script defer src="b.js">
顺序：保证先后顺序。解析：HTML 解析器遇到它们时，不阻塞（脚本将被异步下载），
待文档解析完成之后，执行脚本。
```

#### 3.尽快执行
```
<script async src="a.js">
<script async src="b.js">
顺序：不保证先后顺序。解析：HTML 解析器遇到它们时，不阻塞（脚本将被异步下载，一旦下载完成，立即执行它），
并继续解析之后的文档。
```
